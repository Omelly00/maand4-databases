# Notities week 13 - Databases

## Database

Een database is een plek waar je gegevens gestructureerd opslaat.

## Tabel

Een tabel is een onderdeel van een database.  
Je kunt het vergelijken met een Excel-tabblad.

## Rij

Een rij is één record.  
Bijvoorbeeld één student.

## Kolom

Een kolom is een eigenschap.  
Bijvoorbeeld naam, leeftijd of opleiding.

## SQL

SQL is de taal waarmee je met een database praat.

## PostgreSQL

PostgreSQL is een databasesysteem.

## psql

psql is de terminaltool waarmee je PostgreSQL kunt bedienen.

## SELECT

Met SELECT vraag je gegevens op.

Voorbeeld:

```sql
SELECT * FROM studenten;
